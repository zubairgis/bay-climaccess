# Version 3.0 redesign

- Reframed the portal around the Nexa Proof-of-Concept objective: climate-responsive malaria service delivery.
- Separated live Google Earth Engine analytics from uploaded public evidence maps.
- Removed live forecast and Sentinel-1 processing from the GitHub map dropdown.
- Added the live Sentinel-1 flood-detection URL.
- Replaced household/building-level operational attributes with settlement and facility-catchment units.
- Added WorldPop age–sex population evidence while clarifying that WorldPop does not identify pregnancy directly.
- Separated government, modelled and surveyed health evidence.
- Added a human-confirmed watch-to-recovery trigger protocol.
- Added public schemas for population, health observations, facility readiness, climate-risk events, accessibility, hotspots and actions.
- Added a separate evidence-confidence concept rather than hiding data quality inside the priority score.
- Restricted names, phone numbers, household records, calls, referrals and individual health information to a secure system outside GitHub.
