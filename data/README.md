# Data folder

The public analytical units are **settlements**, **health-facility catchments**, and **event/month summaries**.

## Reference
Stable identifiers, boundaries, settlements, facilities and catchment assignments.

## Observations
Long-format templates that preserve government, modelled, survey, sensor and forecast evidence separately. Raw restricted records should remain in a secure database and only approved aggregates should be exported here.

## Outputs
GeoJSON/CSV files suitable for the public map and tables: population summaries, accessibility, service continuity, readiness, hotspots, validation and verified actions.

## Metadata
Indicator definitions, source types, evidence grades and the data dictionary.

Never upload names, phone numbers, exact vulnerable-household coordinates, pregnancy registers, child registers, individual malaria results, call logs, referral records, passwords or credentials.
